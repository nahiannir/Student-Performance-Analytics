## 🚀 Overview

A clean and beginner-friendly data analysis project that explores student performance using Pandas & Jupyter Notebook.

This project demonstrates:

- 📊 Data analysis workflow
- 🧠 Feature engineering
- 📈 Visualization
- 💡 Insight generation

## 🧠 Key Insights

- 🏆 Top performers show consistency across all subjects
- 📉 Lower performers struggle mainly in science subjects
- 📚 English scores are generally higher
- 📊 Overall class performance is moderately distributed
